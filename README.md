# DrmPR

| Version | Supported          |
| ------- | ------------------ |
| Mobile   | :white_check_mark: |
| Desktop   | :white_check_mark: |

## Required
- Nodejs
- Curl
- Python

## Setup
1.Install
```sh
$ npm install
```
2.Setup config
```sh
$ npm setup
```
3.Setup token bot
```sh
$ npm token
```
4.Start Bot
```sh
in Linux
$ cd bin
$ ./dotbot.sh

in windows
$ cd bin
$ dotbot
```
